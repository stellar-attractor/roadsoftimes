## Дизайн-система

### Цветовая палитра (из мокапа)
```css
--rot-bg-deep:    #0f0e09;   /* фон страницы — почти чёрный с охрой */
--rot-bg-card:    #1a1a0e;   /* фон карточек */
--rot-bg-panel:   #232318;   /* панели, хедер */
--rot-border:     #3a3520;   /* границы, разделители */
--rot-gold:       #c9a84c;   /* золотой акцент — заголовки, иконки */
--rot-gold-dim:   #8a6f2e;   /* приглушённое золото */
--rot-text:       #c8c0a8;   /* основной текст */
--rot-text-dim:   #7a7060;   /* вторичный текст, даты */
--rot-red:        #8b2020;   /* акцент — теги "Артефакт", hover */
--rot-tag-exp:    #1a3a2a;   /* фон тега "Экспедиция" (тёмно-зелёный) */
--rot-tag-mus:    #1a2a3a;   /* фон тега "Музей" (тёмно-синий) */
--rot-tag-art:    #3a1a1a;   /* фон тега "Артефакт" (тёмно-красный) */
```

### Типографика
- Заголовки: **Oswald** (Google Fonts) — уже подключён в GameTown
- Основной текст: **PT Sans** (Google Fonts) — уже подключён в GameTown
- Акцентные подписи: **Courier New** или **PT Mono** — для геолокации, цитат

### Текстура фона
- Тёмная зернистая текстура через CSS (noise filter или SVG data URI) — без внешних картинок
- Альтернатива: загрузить `GameTown Blogger Template/images/background.jpg` в Blogger и переиспользовать с overlay

## Этапы разработки

### Этап 1 — HTML-превью (локальная верстка)
_Цель: сверстать всю страницу в чистом HTML/CSS без Blogger, чтобы быстро итерировать_

- [ ] **1.1** Создать `preview/index.html` — скелет страницы с реальными фото
- [ ] **1.2** Хедер: логотип-текст "ДОРОГИ ВРЕМЁН" + nav + иконки поиска/меню
- [ ] **1.3** Hero-слайдер: jQuery Cycle, 3 слайда из имеющихся фото, текст + CTA-кнопки + геолокация
- [ ] **1.4** Сетка "Последние публикации": 3 карточки (фото + бейдж + заголовок + текст + мета)
- [ ] **1.5** Правый сайдбар: заглушки для Карты / "Сейчас в пути" / Цитаты / Тем / Подписки
- [ ] **1.6** Блок "Рекомендуемые музеи": горизонтальная карточка
- [ ] **1.7** Блок "Выберите свой путь": 4 иконки-категории
- [ ] **1.8** Футер: логотип, 3 колонки навигации, соцсети
- [ ] **1.9** Адаптация под мобильный (базовая — 1 колонка)

### Этап 2 — Blogger XML шаблон
_Цель: перенести вёрстку в валидный Blogger XML_

- [ ] **2.1** Создать `template/roadsoftimes.xml` — взять GameTown как основу скелета
- [ ] **2.2** Заменить `<b:skin>` — вставить всю кастомную CSS из превью
- [ ] **2.3** Перенести хедер (статичный HTML, без Blogger-виджетов навигации — вручную)
- [ ] **2.4** Перенести Hero-слайдер (статичный блок, заполняется вручную)
- [ ] **2.5** Подключить jQuery Cycle из GameTown, адаптировать инициализацию
- [ ] **2.6** Настроить `<b:widget type='Blog'>` — шаблон карточки поста (фото превью, заголовок, отрывок, дата, метки)
- [ ] **2.7** Сайдбар — стандартные Blogger-виджеты: HTML/JavaScript для каждого блока
- [ ] **2.8** Блоки "Рекомендуемые музеи" и "Выберите свой путь" — статичный HTML внутри виджетов
- [ ] **2.9** Футер — статичный HTML
- [ ] **2.10** Настройка `body#layout` — скрыть то что не нужно в редакторе

### Этап 3 — Тестирование в Blogger
_Цель: загрузить XML в реальный Blogger и устранить расхождения_

- [ ] **3.1** Импорт XML в тестовый блог
- [ ] **3.2** Создать 3–5 тестовых постов с метками (Экспедиция / Музей / Артефакт / История)
- [ ] **3.3** Загрузить фото в Blogger, заменить заглушки в слайдере и блоках реальными URL
- [ ] **3.4** Проверить адаптив на мобильном
- [ ] **3.5** Финальная правка CSS по результатам тестирования

### Этап 4 — Контент и полировка
- [ ] **4.1** Финальные тексты для Hero-слайдов (3 слайда с реальными фото)
- [ ] **4.2** Наполнение сайдбара (Карта — статичный скриншот; Цитата; Темы)
- [ ] **4.3** Блок "Рекомендуемые музеи" — первый реальный музей
- [ ] **4.4** Иконки для "Выберите свой путь" — SVG или emoji-based

## Что делается вручную (ограничения Blogger)

| Блок | Как обновлять |
|---|---|
| Hero-слайдер | Правка XML в Edit HTML — менять `<img src>`, заголовок, текст, ссылку |
| Рекомендованный музей | Правка HTML-виджета в Layout |
| Карта экспедиций | Статичная картинка или Google My Maps iframe в HTML-виджете |
| "Сейчас в пути" | Правка HTML-виджета |
| Цитата эпохи | Правка HTML-виджета |
| Геолокация под слайдером | Правка в XML |
