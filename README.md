# Дороги Времён — Blogger Theme

> Авторский блог об истории Европы · Путешествия · Музеи · История

Кастомная тёмная тема для Blogger, разработанная для военно-исторического тревел-блога **[roadsoftimes.blogspot.com](https://roadsoftimes.blogspot.com)**. Построена поверх базового шаблона GameTown методом хирургической замены через Python-скрипт.

## Особенности

### Дизайн
- Тёмная военно-историческая палитра: фон `#0f0e09`, золотой акцент `#c9a84c`
- Типографика: **Oswald** (заголовки/навигация) + **PT Sans** (текст) + **PT Mono** (метаданные)
- Полностью адаптивная вёрстка — три точки перелома: 1024px, 768px, 480px

### Шапка и навигация
- Sticky-шапка с логотипом, навигацией и тулбаром соцсетей
- Автоподсветка активного раздела навигации
- **Бургер-меню** (☰) в конце навбара — выпадающая вертикальная панель с подменю вправо
- Тулбар с иконками VK, Telegram (виджет Blogger, редактируется в Макете)

### Hero-слайдер
- Три сменяющихся полноэкранных слайда с геометкой локации
- Кнопка «Читать ›» автоматически ведёт на последний опубликованный пост (через Feeds API)
- Бегущая строка анонсов поверх слайдера (виджет Blogger, редактируется в Макете)

### Лента постов
- Два режима отображения: **карточки** (3 колонки) и **лента** (список с превью)
- Режим сохраняется в `localStorage`
- Баннер с названием текущей метки на страницах фильтрации

### Страница поста
- Полноширинный контент (занимает всю колонку без сайдбара)
- Поддержка CSS-класса `.rot-avatar` для круглых фото автора
- Тёмная тема для формы комментариев (CSS-инверсия iframe)

### Сайдбар
- Виджет «В пути» (текущий маршрут)
- Виджет «Цитата»
- Облако меток (цвет/размер по частоте)
- Архив с раскрывающимися годами/месяцами

### Футер
- Четыре колонки: описание проекта, навигация, информация, соцсети
- Ссылки VK, Telegram (YouTube — при появлении канала)

## Сборка и деплой

### Требования
- Python 3.8+
- Исходный шаблон GameTown в папке `GameTown Blogger Template/`

### Сборка шаблона

```bash
python3 build_template.py
```

Результат: `template/roadsoftimes.xml`

### Загрузка в Blogger

1. Blogger → Тема → Настроить → Восстановить → загрузить `roadsoftimes.xml`
2. Blogger → Макет → виджет **Тулбар** (HTML11) → вставить содержимое `widgets/header-toolbar.html`
3. Blogger → Макет → виджет **Newsfeed** (HTML12) → вставить содержимое `widgets/hero-newsfeed.html`

### Редактирование меню

Открой `menu_config.py` и измени список `MENU_ITEMS`:

```python
MENU_ITEMS = [
    ('🌍 Страны', '#', [
        ('🇩🇪 Германия', '/search/label/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F'),
        ('🇵🇱 Польша',   '/search/label/%D0%9F%D0%BE%D0%BB%D1%8C%D1%88%D0%B0'),
    ]),
    # ...
]
```

Затем пересобери шаблон: `python3 build_template.py` и загрузи в Blogger.

Формат пункта: `('Название', '/url', [подпункты])`.  
Разделитель: `('---', '', [])`.

| ID     | Название          | Где находится       | Описание                          |
|--------|-------------------|---------------------|-----------------------------------|
| HTML2  | Маршрут           | Сайдбар             | Текущая поездка                   |
| HTML8  | Цитата            | Сайдбар             | Цитата автора                     |
| HTML9  | Рекомендую музей  | Под лентой постов   | Блок с рекомендуемым музеем       |
| HTML10 | *(без названия)*  | Сайдбар             | Метки + Архив (таб-виджет)        |

| HTML12 | Newsfeed          | Слайдер (внизу)     | Бегущая строка анонсов            |

## Технические решения

**Blogger XML + Python-шаблонизатор**  
Шаблон собирается скриптом, а не правится вручную — это позволяет держать логику в читаемом Python-коде и легко вносить изменения.

**Видимость виджетов в Layout-редакторе**  
Виджеты тулбара и ньюсфида живут в скрытом `#floating-widgets` (за экраном), JS перемещает их на место. CSS-overrides с `body#layout` делают их видимыми в редакторе макета.

**Определение типа страницы**  
Blogger не позволяет условно открывать/закрывать div через `<b:if>`. Решение: JS вставляет класс `rot-page-<pageType>` в `<body>`, далее чистый CSS управляет отображением.

**XML-безопасный JS**  
Все скрипты в шаблоне учитывают ограничения XML: `&&` заменены вложенными `if()`, `<` — на `.forEach()`, `&` в URL — на `&amp;`, unicode-escape в CSS через lambda в `re.sub`.
