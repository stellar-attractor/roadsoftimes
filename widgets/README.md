# Widgets — HTML-заготовки для Blogger

Файлы в этой папке — исходный код виджетов, которые вставляются вручную
через редактор Blogger (Макет → карандашик на виджете → вставить HTML).

Они **не собираются автоматически** скриптом `build_template.py`.
CSS-классы для всех виджетов определены в теме.

## Как вставить / обновить

1. Открой нужный `.html` файл, скопируй содержимое
2. Blogger → **Макет** → найди нужный виджет в сайдбаре → **карандашик**
3. Вставь HTML → **Сохранить**

## Файлы

| Файл | Виджет | Blogger ID | Меняется |
|------|--------|------------|----------|
| `sidebar-onroad.html` | Маршрут («В пути») | `HTML2` | При каждой поездке |
| `sidebar-quote.html` | Цитата («От автора») | `HTML_QUOTE` | По настроению |
| `museum-featured.html` | Рекомендуемый музей | `HTML_MUSEUM` | При смене музея |

## Заметки по редактированию

### sidebar-onroad.html
- **Статус «ПАУЗА»**: замени цвет точки `#5a8a5a` → `#8a6f2e` и текст `В ПУТИ` → `ПАУЗА`
- **Статус «В ПУТИ»**: цвет `#5a8a5a` (зелёный), точка пульсирует через CSS-анимацию
- Поля: `rot-onroad-dest` (город), `rot-onroad-country` (страна),
  `rot-onroad-route` (маршрут через стрелки →), `rot-onroad-dates` (период)

### sidebar-quote.html
- Меняй только текст внутри `<p class="rot-quote-text">` и `rot-quote-author`

### museum-featured.html
- Виджет находится **в левой колонке, под постами** (не в сайдбаре)
- При смене музея: новое фото (`src`), локация, название, описание, статистика, ссылка «Подробнее»
